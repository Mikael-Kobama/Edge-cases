# Desafio 09: Validador de Estrutura de CPF

## 📝 Enunciado

Crie um programa que receba uma string contendo um CPF e realize as seguintes validações e formatações:

1. Verifique se a entrada contém exatamente **11 dígitos**.
2. Remova caracteres especiais (pontos e traços) caso existam.
3. Separe os 9 primeiros dígitos dos 2 dígitos verificadores finais.
4. Exiba se o formato é "Válido" (apenas por extensão) ou "Inválido".

_Bônus: Implementar a lógica matemática de validação do primeiro dígito._

## 🛠️ Conceitos Praticados

- Manipulação de Strings (Slicing e `replace`).
- Tratamento de dados de entrada.
- Condicionais aninhadas.

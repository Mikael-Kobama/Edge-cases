# 📊 Contador de Frequência

### Descrição

Escreva um programa que conte a frequência de cada caractere em uma string e retorne os resultados em um dicionário.

### Exemplos

- **Entrada:** `"banana"`
- **Saída:** `{'b': 1, 'a': 3, 'n': 2}`

### Requisitos

- Utilizar estruturas de repetição (`for`).
- Utilizar dicionários para armazenamento.

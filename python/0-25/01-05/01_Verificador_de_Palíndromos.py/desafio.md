# 🧩 Verificador de Palíndromos

### Descrição

Crie uma função que receba uma string e determine se ela é um palíndromo. Um palíndromo é uma palavra ou frase que se lê da mesma forma de trás para frente.

### Regras

- Deve ignorar a diferença entre maiúsculas e minúsculas.
- Deve ignorar espaços em branco.

### Exemplos

| Entrada   | Saída   |
| :-------- | :------ |
| "Arara"   | `True`  |
| "Python"  | `False` |
| "Amo oma" | `True`  |

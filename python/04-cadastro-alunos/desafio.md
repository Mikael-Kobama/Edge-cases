# 🎓 Cadastro de Alunos

### Descrição

Crie um programa que armazene nomes de alunos e suas respectivas notas em um dicionário. O programa deve calcular a média da turma e identificar quem está acima da média.

### Regras

- O usuário pode inserir quantos alunos desejar.
- O programa deve parar quando o usuário digitar "sair".
- Deve exibir a média geral e uma lista com os nomes dos alunos aprovados (nota >= 7.0).

### Tópicos Praticados

- Dicionários (`dict`).
- Loops `while` com condição de parada.
- Manipulação de listas e médias aritméticas.

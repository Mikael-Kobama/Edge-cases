# 🐝 FizzBuzz

### Descrição

Implemente o clássico desafio FizzBuzz para números de 1 a 50.

### Regras

- Para múltiplos de 3, exiba "Fizz".
- Para múltiplos de 5, exiba "Buzz".
- Para múltiplos de ambos, exiba "FizzBuzz".
- Para os demais, exiba o próprio número.

### Objetivos

- Praticar operadores aritméticos (`%`).
- Praticar estruturas condicionais aninhadas ou compostas.

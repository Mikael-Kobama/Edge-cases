# Desafio 08: Jogo da Adivinhação v2.0

## 📝 Enunciado

O computador deve "pensar" em um número inteiro entre **0 e 10**. O jogador deve tentar adivinhar até acertar.

**Regras:**

1. A cada palpite errado, o programa deve dar uma dica: "Mais alto..." ou "Mais baixo...".
2. O jogo só termina quando o usuário acertar.
3. Ao final, exiba o total de tentativas necessárias para vencer.

## 🛠️ Conceitos Praticados

- Importação de bibliotecas (`random`).
- Loops de interrupção dinâmica.
- Operadores de comparação e contadores.

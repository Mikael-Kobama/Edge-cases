# Desafio 07: Analisador de Números

## 📝 Enunciado

Desenvolver um script que leia múltiplos números inteiros fornecidos pelo usuário. O programa deve parar quando o usuário digitar o valor `0`.

Ao final, o programa deve exibir:

- A **quantidade** de números digitados (excluindo o zero).
- A **soma** total dos valores.
- A **média aritmética** dos valores.
- O **maior** e o **menor** valor informado.

## 🛠️ Conceitos Praticados

- Estrutura de repetição `while`.
- Manipulação de listas (`list`).
- Funções nativas: `sum()`, `len()`, `max()` e `min()`.

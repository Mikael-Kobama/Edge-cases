# 🔢 Analisador de Números Únicos

### Descrição

Dada uma lista de números que contém muitos duplicados, crie uma função que retorne apenas os números únicos, ordenados do menor para o maior.

### Exemplos

- **Entrada:** `[10, 2, 2, 5, 10, 1, 5]`
- **Saída:** `[1, 2, 5, 10]`

### Tópicos Praticados

- Conjuntos (`set`) para remoção de duplicatas.
- Conversão de tipos (`list` <-> `set`).
- Método `sorted()`.

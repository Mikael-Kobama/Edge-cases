# Desafio 04: Enums (Categorias Nomeadas)

## 📝 Enunciado

Crie um `enum` chamado `Prioridade` com os valores: `Baixa`, `Media` e `Alta`. Crie uma função que receba essa prioridade e exiba uma mensagem diferente para cada nível.

## 🛠️ Por que usar isso?

Usar strings soltas como `"alta"` ou `"ALTA"` gera erros de digitação. O **Enum** centraliza as opções permitidas, oferecendo autocompletar no VS Code e impedindo valores inválidos.

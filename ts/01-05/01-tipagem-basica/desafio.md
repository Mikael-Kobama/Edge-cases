# Desafio 01: Tipagem Primitiva e Inferência

## 📝 Enunciado

Crie variáveis para armazenar o nome de um curso (string), o valor da mensalidade (number) e se as inscrições estão abertas (boolean).

**Regras:**

1. Declare os tipos explicitamente.
2. Tente reatribuir um número à variável de string e observe o erro de compilação do TSC (TypeScript Compiler).

## 🛠️ Por que usar isso?

No JavaScript comum, uma variável pode mudar de tipo livremente (tipagem dinâmica), o que gera bugs difíceis de rastrear. O TS introduz a **Tipagem Estática**, onde o tipo é verificado antes mesmo do código rodar, garantindo que "mensalidade" sempre seja um número para cálculos matemáticos.

# Desafio 03: Interfaces (Shapes de Objetos)

## 📝 Enunciado

Crie uma `interface` chamada `Usuario` que contenha as propriedades: `nome`, `email` e uma propriedade opcional `telefone`. Em seguida, crie um objeto que siga esse contrato.

## 🛠️ Por que usar isso?

Interfaces definem o **Shape** (formato) dos dados. Elas são essenciais em aplicações grandes para garantir que todos os objetos de "Usuário" no sistema tenham as mesmas propriedades, evitando erros de `undefined` ao tentar acessar chaves inexistentes.

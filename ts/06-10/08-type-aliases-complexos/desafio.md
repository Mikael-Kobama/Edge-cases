# Desafio 08: Type Aliases e Interseção

## 📝 Enunciado

Crie dois tipos: `DadosPessoais` (nome, idade) e `DadosConta` (email, senha). Crie um terceiro tipo chamado `UsuarioCompleto` que seja a **junção** (interseção) dos dois primeiros.

## 🛠️ Por que usar isso?

O operador `&` (Intersection) permite compor tipos como se fossem peças de LEGO. Isso evita a repetição de código e permite criar modelos de dados complexos a partir de modelos simples.

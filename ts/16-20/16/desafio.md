# Desafio 16: Concatenação de Matrizes Simples

Crie uma função que receba dois arrays de números (`number[]`) como parâmetros. A função deve juntar os dois em um único array e retornar o resultado.

**Exigência:** Tipar os dois arrays de entrada e garantir que o retorno seja explicitamente `number[]`.

## Entrada para solucao.ts

```typescript
const listaA: number[] = [1, 2, 3];
const listaB: number[] = [4, 5, 6];
```

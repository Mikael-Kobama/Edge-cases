# Desafio 13: Verificação de Maioria

Crie uma função que receba cinco valores booleanos (v1, v2, v3, v4, v5).
A função deve retornar `true` apenas se 3 ou mais desses valores forem verdadeiros.

**Exigência:** Praticar lógica condicional e repetição de tipos booleanos.

## Entrada para solucao.ts

```typescript
const v1: boolean = true;
const v2: boolean = false;
const v3: boolean = true;
const v4: boolean = false;
const v5: boolean = true;
```

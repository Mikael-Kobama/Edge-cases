# Desafio 11: Operação com 5 Números

Crie uma função que receba cinco parâmetros (n1, n2, n3, n4, n5) do tipo `number`.
A função deve retornar a soma dos dois primeiros multiplicada pela soma dos três últimos.

**Exigência:** Tipagem explícita em todos os parâmetros e no retorno.

## Entrada para solucao.ts

```typescript
const n1: number = 10;
const n2: number = 5;
const n3: number = 2;
const n4: number = 2;
const n5: number = 1;
```

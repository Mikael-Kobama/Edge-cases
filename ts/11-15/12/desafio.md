````markdown
# Desafio 12: Concatenação com Separador

Crie uma função que receba cinco strings (s1, s2, s3, s4, s5) e retorne uma única string onde todas as palavras estão separadas por " | ".

**Exigência:** Uso de Template Literals e tipagem rigorosa de strings.

```typescript
const s1: string = "TS";
const s2: string = "Logica";
const s3: string = "Repeticao";
const s4: string = "Desafio";
const s5: string = "12";
````

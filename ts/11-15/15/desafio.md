````markdown
# Desafio 15: Extração de Interface

Crie uma interface chamada `Filme` com as propriedades `titulo` (string) e `ano` (number).
Crie uma função que receba três objetos do tipo `Filme` e retorne um array de strings contendo apenas os títulos.

**Exigência:** Definição de Interface e tipagem de array de strings no retorno.

## Entrada para solucao.ts

```typescript
interface Filme {
  titulo: string;
  ano: number;
}
```
````

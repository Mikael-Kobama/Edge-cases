# Desafio 14: Filtro de Valores

Crie uma função que receba cinco números como parâmetros individuais.
A função deve processar esses números e retornar um array (`number[]`) contendo apenas os que forem maiores que 10.

**Exigência:** Retorno tipado como array numérico.

## Entrada para solucao.ts

```typescript
const a: number = 5;
const b: number = 12;
const c: number = 8;
const d: number = 20;
const e: number = 7;
```

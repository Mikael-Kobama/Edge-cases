# Desafio 08: Contador de Vogais

## 📝 Enunciado

Crie uma função que receba uma string e retorne um objeto com a contagem de cada vogal (a, e, i, o, u), ignorando maiúsculas/minúsculas.

## 🛠️ Conceitos Praticados

- Loop `for...of` ou `.forEach()`
- Objetos (Chave/Valor)
- String methods (`toLowerCase`)

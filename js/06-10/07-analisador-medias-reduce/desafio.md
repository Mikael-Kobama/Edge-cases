# Desafio 07: Analisador de Médias com Reduce

## 📝 Enunciado

Dado um array de notas, utilize o método `.reduce()` para calcular a média aritmética do aluno.

- Se a média for >= 7: "Aprovado"
- Se a média for < 7: "Reprovado"

## 🛠️ Conceitos Praticados

- Array.prototype.reduce()
- Lógica condicional (Ternário)

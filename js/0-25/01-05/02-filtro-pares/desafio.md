# 🔢 Filtro de Números Pares

### Descrição

Dada uma lista (array) de números inteiros, crie uma função que retorne um novo array contendo apenas os números que são pares.

### Exemplos

- **Entrada:** `[1, 2, 3, 4, 5, 6]`
- **Saída:** `[2, 4, 6]`

### Requisitos

- Utilizar o método de alta ordem `.filter()`.
- Utilizar o operador de resto de divisão `%`.

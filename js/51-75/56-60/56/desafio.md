# 56-formatacao-lista-com-id-personalizado

![Nível: Médio](https://img.shields.io/badge/Nível-Médio-yellow?style=flat-square)

### 📝 Descrição

Receba um array de nomes e transforme-os em objetos, onde cada objeto tem um `id` único (baseado no índice) e o `nome` com a primeira letra em maiúscula.

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.map()`.
- [x] Manipulação de String (`charAt` + `slice`).
- [x] Criação de objeto dinâmico.

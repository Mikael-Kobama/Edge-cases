# 58-limpeza-de-dados-duplicados

![Nível: Médio+](https://img.shields.io/badge/Nível-Médio+-orange?style=flat-square)

### 📝 Descrição

Receba uma lista de IDs de transações da AWS que pode conter valores repetidos e retorne uma nova lista apenas com valores únicos.

### 🛠️ Requisitos Técnicos

- [x] Utilizar a estrutura de dados `Set`.
- [x] Utilizar o **Spread Operator** `[...]` para converter de volta para Array.

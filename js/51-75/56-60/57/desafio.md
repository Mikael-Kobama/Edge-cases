# 57-filtro-e-ordenacao-por-preco

![Nível: Médio](https://img.shields.io/badge/Nível-Médio-yellow?style=flat-square)

### 📝 Descrição

Dada uma lista de produtos, filtre apenas os que custam mais de R$ 100,00 e ordene-os do **mais barato para o mais caro**.

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.filter()`.
- [x] Utilizar `.sort()` (Cuidado: o sort altera o array original, use o spread `[...]` para evitar isso).

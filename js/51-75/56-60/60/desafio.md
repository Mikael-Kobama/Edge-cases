# 60-super-pipeline-de-relatorio-financeiro

![Nível: Difícil](https://img.shields.io/badge/Nível-Difícil-red?style=flat-square)

### 📝 Descrição

Processamento completo de folha de pagamento:

1. **Filtrar** colaboradores do setor "TI".
2. **Mapear** adicionando um bônus de 10% ao salário.
3. **Mapear** formatando o nome para: "DEV: [NOME]".
4. **Reduzir** para calcular o custo total da folha desse setor.

### 🛠️ Requisitos Técnicos

- [x] Encadeamento de 4 métodos (Filter -> Map -> Map -> Reduce).
- [x] Lógica de bônus percentual.

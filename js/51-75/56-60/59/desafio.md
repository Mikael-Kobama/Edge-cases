# 59-agrupamento-por-status-booleano

![Nível: Médio+](https://img.shields.io/badge/Nível-Médio+-orange?style=flat-square)

### 📝 Descrição

Dada uma lista de tarefas, separe-as em um objeto com duas chaves: `concluidas` e `pendentes`.

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.reduce()`.
- [x] Inicializar o acumulador com a estrutura de objeto correta.

# 68-analise-de-vendas-por-periodo

![Nível: Difícil](https://img.shields.io/badge/Nível-Difícil-red?style=flat-square)

### 📝 Descrição

Dada uma lista de vendas com datas, você deve:

1. Filtrar as vendas realizadas no ano de 2026.
2. Agrupar o total vendido por categoria (ex: "Eletrônicos", "Móveis").
3. Retornar um objeto onde a chave é a categoria e o valor é o total somado.

### 🛠️ Requisitos Técnicos

- [x] Manipulação de `new Date().getFullYear()`.
- [x] `.reduce()` com acumulador dinâmico.

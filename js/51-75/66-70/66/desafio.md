# 66-validador-e-totalizador-de-carrinho

![Nível: Médio+](https://img.shields.io/badge/Nível-Médio+-orange?style=flat-square)

### 📝 Descrição

Dado um carrinho de compras, você deve:

1. Filtrar apenas itens que possuem preço e quantidade válidos (maiores que zero).
2. Calcular o subtotal de cada item (preço \* quantidade).
3. Retornar um objeto com o `valorTotal` e uma lista dos `nomes` dos produtos aprovados.

### 🛠️ Requisitos Técnicos

- [x] `.filter()` + `.reduce()`.
- [x] Criação de objeto de retorno complexo.

# 67-gerador-de-crachas-com-limpeza

![Nível: Médio+](https://img.shields.io/badge/Nível-Médio+-orange?style=flat-square)

### 📝 Descrição

Receba uma lista de convidados para um evento tech. Você deve:

1. Remover duplicatas (mesmo nome).
2. Filtrar nomes vazios ou que contenham apenas espaços.
3. Formatar o nome para "Título: [NOME EM MAIUSCULO]".
4. Ordenar alfabeticamente.

### 🛠️ Requisitos Técnicos

- [x] `new Set()` para duplicatas.
- [x] `.filter()` + `.map()` + `.sort()`.

# 73-rastreador-de-estoque-critico

![Nível: Difícil](https://img.shields.io/badge/Nível-Difícil-red?style=flat-square)

### 📝 Descrição

Você tem uma lista de depósitos, e cada um tem um array de produtos.

1. Use `.flatMap()` para unificar todos os produtos em uma única lista.
2. Filtre produtos com estoque menor que 10.
3. **Reforço:** Verifique se o produto já existe na lista final (pelo nome). Se sim, **some** as quantidades em vez de duplicar o item.
4. Retorne a lista de reposição urgente.

### 🛠️ Requisitos Técnicos

- [x] `.flatMap()`, `.reduce()`.
- [x] Lógica de busca em acumulador.

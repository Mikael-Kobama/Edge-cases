# 72-calculadora-de-comissao-escalonada

![Nível: Médio+](https://img.shields.io/badge/Nível-Médio+-orange?style=flat-square)

### 📝 Descrição

Calcule a comissão de uma equipe de vendas:

1. Vendas até R$ 1000: 5% de comissão.
2. Vendas acima de R$ 1000: 10% de comissão.
3. **Reforço:** Adicione um bônus fixo de R$ 50 para quem fez mais de 3 vendas (independente do valor).
4. Retorne um objeto com o `nome` e o `totalComissao`.

### 🛠️ Requisitos Técnicos

- [x] `.map()` para transformar o objeto.
- [x] Lógica condicional (If/Else ou Ternário).

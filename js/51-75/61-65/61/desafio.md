# 61-mapeamento-de-entidades-objeto

![Nível: Médio](https://img.shields.io/badge/Nível-Médio-yellow?style=flat-square)

### 📝 Descrição

Converta um objeto de "Preços de Serviços Cloud" em uma lista de frases legíveis, utilizando as chaves e os valores.

### 🛠️ Requisitos Técnicos

- [x] Utilizar `Object.entries()`.
- [x] Utilizar `.map()` com desestruturação de array `([chave, valor])`.

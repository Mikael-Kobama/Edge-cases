# 64-validador-de-logs-regex

![Nível: Difícil](https://img.shields.io/badge/Nível-Difícil-red?style=flat-square)

### 📝 Descrição

Dada uma lista de logs de servidor, filtre apenas as linhas que contêm erros críticos (Status 500 ou 404) e extraia apenas a mensagem de erro.

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.filter()` com `.test()` de Regex.
- [x] Utilizar `.map()` com `.replace()` ou `.split()`.

# 63-ordenacao-multi-criterio

![Nível: Médio+](https://img.shields.io/badge/Nível-Médio+-orange?style=flat-square)

### 📝 Descrição

Ordene uma lista de colaboradores primeiro pelo **setor** (ordem alfabética) e, em caso de empate, pelo **salário** (do maior para o menor).

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.sort()`.
- [x] Lógica de comparação composta (`||`).
- [x] Método `.localeCompare()` para strings.

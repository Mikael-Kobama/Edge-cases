# 62-extracao-de-subitens-flatmap

![Nível: Médio+](https://img.shields.io/badge/Nível-Médio+-orange?style=flat-square)

### 📝 Descrição

Dado um array de projetos, onde cada projeto tem uma lista de tecnologias, retorne uma lista única (flat) com todas as tecnologias usadas na empresa, sem duplicatas.

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.flatMap()`.
- [x] Utilizar `new Set()` para remover duplicatas.

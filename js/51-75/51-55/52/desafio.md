# 52-filtro-data-vencimento

![Nível: Médio](https://img.shields.io/badge/Nível-Médio-yellow?style=flat-square)

### 📝 Descrição

Filtre uma lista de assinaturas de software e retorne apenas aquelas que já venceram (data de vencimento menor que a data atual).

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.filter()`.
- [x] Comparação de instâncias de `new Date()`.

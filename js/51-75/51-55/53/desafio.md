# 53-formatacao-moeda-internacional

![Nível: Médio](https://img.shields.io/badge/Nível-Médio-yellow?style=flat-square)

### 📝 Descrição

Dada uma lista de preços em números, converta-os para uma lista de strings formatadas no padrão de moeda Brasileira (R$).

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.map()`.
- [x] Utilizar o módulo global `Intl.NumberFormat`.

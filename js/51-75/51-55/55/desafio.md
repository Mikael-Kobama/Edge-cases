# 55-busca-e-extracao-com-fallback

![Nível: Médio+](https://img.shields.io/badge/Nível-Médio+-orange?style=flat-square)

### 📝 Descrição

Busque um desenvolvedor específico pelo ID. Se encontrar, retorne uma string com o nome dele e as tecnologias que domina (unidas por vírgula). Se não encontrar, retorne "Usuário não encontrado".

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.find()`.
- [x] Utilizar `.join()` para transformar o array de techs em string.
- [x] Lógica condicional de fallback.

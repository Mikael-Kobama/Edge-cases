# 54-contagem-por-categoria

![Nível: Médio+](https://img.shields.io/badge/Nível-Médio+-orange?style=flat-square)

### 📝 Descrição

Dada uma lista de tickets de suporte com diferentes categorias, retorne um objeto que conte quantos tickets existem por categoria.

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.reduce()`.
- [x] Lógica de acumulador dinâmico (Hash Map).

# 51-acesso-propriedade-aninhada

![Nível: Médio](https://img.shields.io/badge/Nível-Médio-yellow?style=flat-square)

### 📝 Descrição

Dado um array de usuários onde o endereço é um objeto interno, retorne uma lista de strings contendo apenas a cidade de cada usuário.

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.map()`.
- [x] Acessar propriedade de objeto aninhado (`usuario.endereco.cidade`).

# 🔄 Inversor de Strings

### Descrição

Crie uma função que receba uma string e retorne a mesma string, porém invertida.

### Regras

- Não utilize loops `for` manuais se quiser um desafio extra (tente métodos de Array).
- O resultado deve manter a mesma capitalização original das letras.

### Exemplos

- **Entrada:** `"Desenvolvedor"`
- **Saída:** `"rodovlovneseD"`

### Tópicos Praticados

- Métodos de String (`split`).
- Métodos de Array (`reverse`, `join`).

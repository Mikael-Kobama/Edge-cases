# 🏆 Localizador de Maior Número

### Descrição

Escreva uma função que receba um array de números e retorne qual é o maior valor presente nele.

### Exemplos

- **Entrada:** `[5, 12, 8, 130, 44]`
- **Saída:** `130`

### Tópicos Praticados

- Objeto Global `Math`.
- Operador Spread (`...`) ou loops de comparação.

````markdown
# 78-agrupamento-inverso-de-tags

### 📝 Descrição

Dada uma lista de tarefas com tags, crie um **índice remissivo**.
Ao invés de ver a tarefa e suas tags, queremos ver a **tag e quais tarefas pertencem a ela**.
_Use apenas `reduce` e loops internos._

### 📥 Exemplo de Saída Esperada

```json
{
  "bug": ["Corrigir Login", "Erro S3"],
  "feat": ["Login Social"],
  "ui": ["Corrigir Login"]
}
```
````

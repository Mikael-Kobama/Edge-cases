# 🔎 Verificador de Item em Lista

![Nível: Fácil](https://img.shields.io/badge/Nível-Fácil-green?style=flat-square)

### 📝 Descrição

Crie uma função que receba um array de strings e uma palavra, retornando `true` se a palavra estiver na lista.

### 🛠️ Requisitos Técnicos

- [x] Utilizar o método `.includes()`.

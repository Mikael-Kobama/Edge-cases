# Desafio 04: Filtragem de Dados (Maioridade)

## 📝 Enunciado

Dado um array estático de idades, utilize métodos funcionais para criar um novo array contendo apenas as pessoas que possuem **18 anos ou mais**.

**Dados de entrada:** `[12, 18, 22, 10, 30, 15, 65, 17]`

## 🛠️ Conceitos Praticados

- Método de Array `.filter()`.
- Arrow Functions (`=>`).
- Imutabilidade (não alterar o array original).

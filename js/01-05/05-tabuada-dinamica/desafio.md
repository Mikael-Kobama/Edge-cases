# Desafio 05: Tabuada Interativa

## 📝 Enunciado

Crie um script que receba um número via entrada de usuário e gere a tabuada completa de 1 a 10.

A saída deve ser formatada rigorosamente como:
`N x 1 = R`
`N x 2 = R`

## 🛠️ Conceitos Praticados

- Laços de repetição (`for`).
- Template Literals (Interpolação com `${}`).
- Conversão de tipos (`Number()`).

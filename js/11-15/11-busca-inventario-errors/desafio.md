# Desafio 11: Busca no Inventário (Try/Catch)

## 📝 Enunciado

Crie um sistema que busque produtos em um array de objetos. Caso o produto não exista, o sistema deve lançar um erro personalizado e tratá-lo com `try...catch`.

## 🛠️ Conceitos Praticados

- Método `.find()`.
- Tratamento de exceções (`throw new Error`).

# 🔍 Verificador de Propriedade em Objeto

![Nível: Fácil](https://img.shields.io/badge/Nível-Fácil-green?style=flat-square)

### 📝 Descrição

Crie uma função que receba um objeto e uma string (chave), e retorne `true` se a propriedade existir no objeto, ou `false` caso contrário.

### 🛠️ Requisitos Técnicos

- [x] Utilizar o método `.hasOwnProperty()` ou o operador `in`.

# 44-validador-extensao-segura

![Nível: Médio](https://img.shields.io/badge/Nível-Médio-yellow?style=flat-square)

### 📝 Descrição

Verificar se todos os arquivos em uma lista possuem a extensão `.js` para garantir a segurança do deploy.

### 🛠️ Requisitos Técnicos

- [x] Utilizar o método `.every()`.
- [x] Utilizar o método de string `.endsWith()`.

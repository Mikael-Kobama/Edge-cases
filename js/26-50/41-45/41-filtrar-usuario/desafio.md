# 🟢 Filtro de Status Ativo

![Nível: Fácil](https://img.shields.io/badge/Nível-Fácil-green?style=flat-square)

### 📝 Descrição

O objetivo é filtrar uma lista de objetos de usuários e retornar apenas aqueles que possuem a propriedade `ativo` como `true`.

### 🛠️ Requisitos Técnicos

- [x] Utilizar o método `.filter()`.
- [x] Retornar o objeto completo do usuário.

### 📥 Entrada

```javascript
[
  { nome: "Mikael", ativo: true },
  { nome: "Dev", ativo: false },
];
```

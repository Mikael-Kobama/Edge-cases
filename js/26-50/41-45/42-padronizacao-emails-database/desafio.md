# 📧 Padronização de E-mails

![Nível: Fácil](https://img.shields.io/badge/Nível-Fácil-green?style=flat-square)

### 📝 Descrição

Transformar uma lista de e-mails brutos (com letras maiúsculas e minúsculas misturadas) em um padrão totalmente minúsculo para consistência em bancos de dados.

### 🛠️ Requisitos Técnicos

- [x] Utilizar o método `.map()`.
- [x] Utilizar o método de string `.toLowerCase()`.

# 45-pipeline-checkout-ecommerce

![Nível: Médio+](https://img.shields.io/badge/Nível-Médio+-orange?style=flat-square)

### 📝 Descrição

Processar um carrinho de compras seguindo três etapas obrigatórias: filtrar itens caros, aplicar taxa e somar o total.

### 🛠️ Requisitos Técnicos

- [x] **Módulo 1 (.filter):** Filtrar itens acima de R$ 50,00.
- [x] **Módulo 2 (.map):** Adicionar taxa de R$ 10,00.
- [x] **Módulo 3 (.reduce):** Somar o valor final.

# 47-limpeza-lista-strings

![Nível: Fácil](https://img.shields.io/badge/Nível-Fácil-green?style=flat-square)

### 📝 Descrição

Receber uma lista de tags de busca que podem conter espaços extras e retornar uma nova lista com os textos limpos (sem espaços nas pontas).

### 🛠️ Requisitos Técnicos

- [x] Utilizar o método `.map()`.
- [x] Utilizar o método `.trim()`.

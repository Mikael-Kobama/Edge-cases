# 48-filtro-numeros-pares

![Nível: Fácil](https://img.shields.io/badge/Nível-Fácil-green?style=flat-square)

### 📝 Descrição

Dada uma lista de IDs gerados pelo sistema, filtre e retorne apenas os números que são pares.

### 🛠️ Requisitos Técnicos

- [x] Utilizar o método `.filter()`.
- [x] Utilizar o operador de módulo `%`.

# 50-relatorio-final-vendas-cloud

![Nível: Médio+](https://img.shields.io/badge/Nível-Médio+-orange?style=flat-square)

### 📝 Descrição

Processar um relatório de faturamento de serviços Cloud:

1. **Filtrar** apenas faturamentos acima de 100 dólares.
2. **Mapear** convertendo o valor para Real (multiplicar por 5).
3. **Reduzir** somando o valor total em Reais.

### 🛠️ Requisitos Técnicos

- [x] **Módulo 1 (.filter)**
- [x] **Módulo 2 (.map)**
- [x] **Módulo 3 (.reduce)**

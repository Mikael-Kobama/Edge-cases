# 📁 Separador de Arquivos (Simples)

### 📝 Descrição

Dada uma lista de nomes de arquivos, retorne um objeto que os separe em dois arrays: `imagens` (se terminar em .jpg ou .png) e `outros`.

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.reduce()` para criar o objeto final.
- [x] Utilizar `.endsWith()` para verificar a extensão.
- [x] Condicional `if/else`.

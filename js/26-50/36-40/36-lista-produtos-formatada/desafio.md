# 🛒 Lista de Produtos Formatada

![Nível: Médio](https://img.shields.io/badge/Nível-Médio-yellow?style=flat-square)

### 📝 Descrição

Dada uma lista de objetos de produtos (nome e preço), retorne um novo array de strings onde cada item é uma frase formatada: `"O produto [NOME] custa R$ [PREÇO]"`.

### 🛠️ Requisitos Técnicos

- [x] Utilizar o método `.map()`.
- [x] Utilizar **Template Strings** (crases `` ` ``).
- [x] Garantir que o preço tenha duas casas decimais (`.toFixed(2)`).

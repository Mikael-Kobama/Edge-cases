# ⚠️ Filtro de Estoque Crítico

### 📝 Descrição

Filtre uma lista de produtos e retorne apenas os nomes daqueles que possuem **menos de 5 unidades** no estoque.

### 🛠️ Requisitos Técnicos

- [x] Utilizar `.filter()` para a regra de negócio.
- [x] Utilizar `.map()` para extrair apenas o nome (limpeza de dados).
- [x] Combinar os dois métodos em uma única linha (encadeamento).

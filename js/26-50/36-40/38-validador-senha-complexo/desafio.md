# 🔐 Validador de Senha Complexo

### 📝 Descrição

Crie uma função que valide se uma senha é forte. Ela deve retornar `true` apenas se:

1. Tiver pelo menos 8 caracteres.
2. Tiver pelo menos um número.

### 🛠️ Requisitos Técnicos

- [x] Propriedade `.length`.
- [x] Método de string `.search()` ou Regex `/\d/` (para encontrar números).
- [x] Operadores lógicos `&&` (AND).

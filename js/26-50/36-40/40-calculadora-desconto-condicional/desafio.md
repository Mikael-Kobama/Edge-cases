# 💸 Calculadora de Desconto Progressivo

### 📝 Descrição

Crie uma função que calcule o valor final de uma compra:

1. Se o valor for acima de R$ 100,00, aplique 10% de desconto.
2. Se for abaixo, não aplica desconto.

### 🛠️ Requisitos Técnicos

- [x] Lógica de desconto (`valor * 0.9`).
- [x] Retorno formatado como número.
- [x] Uso de Operador Ternário para reduzir linhas.

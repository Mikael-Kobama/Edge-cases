# Desafio 12: Agrupamento de Categorias (Data Transformation)

## 📝 Enunciado

Transforme um array de usuários em um objeto agrupado por categoria.
Entrada: `[{nome: 'A', tipo: 'Adm'}, {nome: 'B', tipo: 'User'}]`
Saída: `{ Adm: ['A'], User: ['B'] }`

## 🛠️ Conceitos Praticados

- Redução de array para objeto.
- Manipulação dinâmica de propriedades.
